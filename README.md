## Uso e Documentação

O QuickReceitas é uma plataforma web que permite aos usuários:

- Encontrar receitas através de busca inteligente
- Compartilhar suas próprias receitas
- Salvar receitas favoritas
- Criar listas de compras
- Seguir outros cozinheiros
